# React Countdown

>React countdown calculate the difference between current date and a date in the future and the holidays between two dates

# Technology Used:

>React js

>Bulma CSS Framework

## Quick Start:

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start

# Build for production
npm run build
```
